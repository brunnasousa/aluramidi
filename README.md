## 💻 Sobre o projeto

- o usuario pode clicar no botão ou pertar a tecla 'Enter' ou 'Space' do teclado.
- Vai ser imitido um som como: Pom, Clap, Tim, Puff, Splash, Toim, Psh, Tic e Tom.
